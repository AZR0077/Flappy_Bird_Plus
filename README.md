# Flappy_Bird_Plus
This is a recreation of the well-known mobile game Flappy Bird with added mechanics.

# Made in JavaScript and HTML

Contains seven files:
  - Assets folder
  - JS game code folder
  - CSS file
  - Gamelab API file
  - HTML website file
  - Two p5play engine files
